各4箇切り替えれば
ライトモードとダークモード切り替えることが可能です。
プログラムで動的制御が必要。

```html
<!-- Light Mode -->
<body>
<!-- Dark Mode -->
<body class="body-dark">

<!-- Light Mode -->
<input type="text" placeholder="E Mail Address"/>
<!-- Dark Mode -->
<input class="input-dark" type="text" placeholder="E Mail Address"/>

<!-- Light Mode -->
<input type="password" placeholder="Password"/>
<!-- Dark Mode -->
<input class="input-dark" type="password" placeholder="Password"/>

<!-- Light Mode -->
<button type="button"><i class="fas fa-key"></i> Login</button>
<!-- Dark Mode -->
<button class="button-dark" type="button"><i class="fas fa-key"></i> Login</button>
```
